Basic information / 基本说明

MasCN stands for Masons, a free-to-use and freely distributed operating system kernel that is open source using the GNU GPLv2.0 license.

MasCN is an embedded operating system kernel developed using C & assembly language. It has a main branch that runs under x86-64 CPUs and a secondary branch that runs under ARMCPU, with consistent functionality between the two architectures.

The function will continue to be developed and improved, and the specific updates will be displayed on the Version page

Development ingestion / 开发引入

The boot part of the system is developed based on UEFI and its interfaces. For example, all the code in the [MasCNBootPkg] section is based on UEFI.

For specific UEFI development instructions, you can check the content of the Docs page or go to the official website of UEFI.

In the actual code development, the source code used in UEFI is in the edk2 project, and the bootstrapper is compiled under the compilation tools provided under edk2.

When you start the system, the action after the boot animation starts to jump out of UEFI and into the kernel.
